# Comandos del sistema

```
pwd         muestra tu ubicacion
ls          lista archivos
mkdir       crea carpeta
rm          barra archivo/carpeta
cp          copia archivos
mv          mueve o renombra
touch       crea archivos vacio
cat         muestra archivo
nano        editor de texto
clear       limpiar terminal   
man         muestra ayuda/manual
```
